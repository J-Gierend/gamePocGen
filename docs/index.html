The Concept

"What if we could generate unique, engaging idle games while we sleep?"

GamePocGen uses a team of specialized AI agents to generate complete incremental games from scratch. Each game is unique, playable in the browser, and designed using proven engagement patterns.

Psychology-Driven Design

The Psychology Reviewer agent evaluates every game design against seven key engagement principles. These are the secret sauce that separates forgettable games from "just one more minute" addictive experiences.

üéØ Agency vs Randomness

Players must feel their decisions matter. Too much RNG feels unfair; too little feels deterministic.

‚è±Ô∏è Reveal Speed

New content appears at the perfect pace - not so fast it overwhelms, not so slow it bores.

üëÅÔ∏è Reward Visibility

Progress must be obvious. Numbers going up, bars filling, unlocks appearing - constant visual feedback.

üò¨ Near-Miss Frequency

Almost reaching a goal creates tension and motivation. "Just 50 more gold!" keeps players engaged.

üé∞ Variable Rewards

Unpredictable bonuses trigger dopamine. Critical hits, lucky drops, bonus multipliers.

üíî Loss Aversion Balance

Stakes make choices meaningful, but punishment shouldn't feel unfair. Risk without frustration.

üí™ Competence Building

Players should feel they're getting better, mastering systems, making smarter decisions over time.

The Generation Process

From idea to playable game in 5 phases. Each phase builds on the previous, with verification steps ensuring quality at every stage.

Idea Generation ~10 min

A creative AI generates a unique game concept by combining patterns (mining + tower building?), themes (underwater? steampunk? cosmic?), and hooks (what makes THIS game special?).

Game Design Document ~60 min

Six specialized agents work in parallel to flesh out every aspect of the design: currencies, progression, prestige, skill trees, UI/UX. The Psychology Reviewer then audits everything for engagement.

Implementation Planning ~15 min

The design is converted into a phased build plan with clear milestones and verification criteria. Each feature is broken into testable chunks.

Game Building ~120 min

An orchestrator AI builds the game using pre-made components: currency systems, upgrade buttons, progress bars, skill trees. Each feature is tested before moving to the next.

Deployment ~5 min

The finished game is automatically deployed to its own subdomain with SSL. Instantly shareable and playable in any browser.
